### Laudanum

You can get it [here](https://github.com/jbarcia/Web-Shells/tree/master/laudanum), or 
```shell
sudo apt install laudanum
```
then
```sh
$ laudanum

/usr/share/laudanum
├── asp
├── aspx
├── cfm
├── helpers
├── jsp
├── php
└── wordpress
```

When you enter `laudanum` it will automatically direct you to `/usr/share/laudanum`
